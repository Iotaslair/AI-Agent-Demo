# Overview
This repo is meant to serve as a gentle introduction into Agents.

It comes with a prompt and a Jira card we want the agent to create

# Getting Started
1. [Install Go](https://go.dev/dl/) or whatever programming language you want it to use
  1. If you're going with Go install [Staticcheck as well](https://staticcheck.dev/docs/getting-started)
2. [Install Opencode](https://opencode.ai/download)
3. Set Opencode to plan mode
4. Copy the text in prompt.md and paste it into opencode
5. Iterate with the AI in plan mode then send it away to go write the code
